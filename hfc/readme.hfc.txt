 SDOSS-HFC wrapper library
 ==========================
 X.Bonnin, 03-MAR-2014

 This directory contains the wrapper library
 to run the SDOSS code in the framework of the 
 HFC.

 It contains the following items :

	/devtest	dev. version of the SDOSS-HFC library.

	/prod		prod. version of the SDOSS-HFC library.

	readme.hfc.txt	current file.
	
	howto.hfc.txt	How to use this library
